Реализовать структуру данных «множество» на основе бинарного дерева поиска (https://ru.wikipedia.org/wiki/Двоичное_дерево_поиска).

Структура данных должна:
* Реализовывать интерфейс ISet (https://docs.microsoft.com/ru-ru/dotnet/api/system.collections.generic.iset-1?view=net-5.0)
* Поддерживать сохранение в файл и загрузку из файла (можно использовать встроенный механизм сериализации)
* Быть покрыта юнит-тестами
